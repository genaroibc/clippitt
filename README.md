# 🎥 Clippitt

## 📜 Description

A tool to create shorts for TikTok, Reels and Youtube shorts uploading a Twitch clip URL.

This project is part of the Cloud Create 2023 hackathon.

You can try it [here](https://clippitt.vercel.app)
